# scratch-bakery
My Scratch project simulates a classic CoolMathGames.com lemonade stand style game. The player runs a bakery and must fulfil pastry orders to receive payment and build up savings. To start, the player is prompted to name their bakery. Once this input is received, the player presses 'space' to start the game and begin taking orders! The customer places an order, which is stored in a "current order" variable. The player then clicks one of the 3 list items in the Pastries list, and the selection is stored in the "selected food" variable; if the selected food matches what the customer ordered, the order is complete and money is added to the bakery's account! Otherwise, the customer waits to receive the correct item.
